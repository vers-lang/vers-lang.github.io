Functions

Function declaration:


fun my_function () {

} ;

You use "fun" to declare a function, parentheses at the end of the name, and a semicolon behind the ending curly brace


External function declaration:

external fun my_external_function () ;

Use "external" for declaring external functions, it would be used for multiple Vers files that are going to be linked together.


Function parameters

fun my_function ( var : string string_argument ) {

Read types for variables, types and values. The function above takes a string argument, it's value is stored in "string_argument".


Variables

Variable declaration:

var : int x = 5 ;

Use "var" to declare a variable, after add it's type, name and value. Types
